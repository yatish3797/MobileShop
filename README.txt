Mobile Shop Coding Exercise
---------------------------

(Non-Docker) Steps to run the project
-------------------------------------
1) Download and extract the ‘Mobile_Shop.zip’ file
2) In the extracted folder ‘Mobile_Shop’, Run the shell script ‘build_and_start_server.sh’. The shell script will build the project and start the tomcat server at ‘8080’ port.


(Docker) Steps to run the project
---------------------------------
1) Download and extract the ‘Mobile_Shop.zip’ file
2) In the extracted folder ‘Mobile_Shop’, Run the shell script ‘docker_build_and_start_server.sh’. The shell script will build the project and start the tomcat server at ‘8080’ port.
